# Python Quiz Game

This is a simple quiz program I created using Python.
It asks some multiple choice questions and gives a timer for each question.
User answers the questions and gets score based on correct answers.

I also used pandas to store the user name and score in a CSV file.

### Features:

* Timer for each question
* Multiple choice questions
* Score calculation
* Stores data in file

### How it works:

* User enters name
* Quiz starts with timer
* Answers are checked
* Score is shown
* Data is saved in `scores.csv`

### Technologies:

* Python
* Pandas
